# 结构化事实清单模板

访谈完成后，输出以下格式的事实清单，作为写初稿的输入。

```markdown
# 事实清单：[文章主题]

## 核心命题
[一句话概括这篇文章要说的一件事]

## 认知翻转
[我以为X，其实Y]

## 读者痛点
[目标读者的什么处境/情绪？]

## 时间线
| 顺序 | 事件 | 时间 | 关键数字 | 用户感受 |
|------|------|------|---------|---------|
| 1 | | | | |
| 2 | | | | |

## 关键事实
- [已确认的事实1]
- [已确认的事实2]

## 外部引用
- [引用内容] — 来源：[来源] — 确认状态：[已确认/用户理解/待验证]

## 涉及的人
- [人物] — 身份：[身份] — 能否写入文章：[能/不能/需匿名]

## 待确认项
- [还没确定的事实]

## 不能写的内容
- [用户明确说不写的]
```
